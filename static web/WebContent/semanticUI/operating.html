Operating System

According to Theaker and Brookes (1993) and Crowley (1997), operating system is simply a program at a fundamental level. It sits on, controls and sufficiently utilizes hardware to help users interact indirectly with other programs and software applications. Tasks undergoing operating system are not entirely visible to users. Operating system must be kept active and constant all the time until the device is shut off. Therefore, each device with certain hardware requires a reliable, suitable and efficient operating system.
The most common operating systems for smartphones can be found on recent market are Android, iOS, BlackBerry OS, Windows Phone, etc.
